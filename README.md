# SLSim Tutorial Notebooks

This repository contains tutorials, examples, and educational resources for the SLSim (Strong Lensing Simulation) pipeline.

The tutorials are designed to help users learn how to simulate strong gravitational lens systems, generate lens populations, create mock observations, and explore the capabilities of SLSim through hands-on Jupyter notebooks.

**Version requirement:** These tutorials are developed for and tested with **SLSim v0.1.1**. We recommend installing this version to ensure compatibility with the notebooks.

The repository serves as a companion to the main SLSim package and provides:

* Getting-started tutorials for new users
* End-to-end simulation examples
* Science-driven use cases and workflows
* Demonstrations of SLSim features and modules
* Educational material for workshops and training events

For the SLSim source code and documentation, see the main SLSim repository:
https://github.com/LSST-strong-lensing/slsim
